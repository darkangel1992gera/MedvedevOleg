# GitHub User Finder

**Автор:** Медведев Олег Игоревич

## Описание

Простое графическое приложение на Python (Tkinter) для поиска пользователей GitHub через публичный API и управления списком избранных. Избранные пользователи сохраняются в локальном JSON-файле и автоматически загружаются при следующем запуске.

Возможности:
- Поиск пользователей по логину.
- Отображение результатов в удобном списке.
- Добавление/удаление пользователей в избранное.
- Просмотр избранного в отдельном окне.
- Валидация ввода (пустое поле блокируется).
- Понятные сообщения об ошибках сети и API.
- Сохранение избранного в `favorites.json`.
- Полное отсутствие внешних зависимостей (только стандартная библиотека Python).

## Инструкция по запуску

1. Убедитесь, что установлен Python 3.8 или выше.
2. Клонируйте репозиторий:
   ```bash
   git clone <URL вашего репозитория>
   cd github-user-finder
   ```
3. Запустите приложение (никакие пакеты не требуются):
   ```bash
   python main.py
   ```

## Как используется GitHub API

Приложение отправляет GET-запрос к публичному эндпоинту:
`https://api.github.com/search/users?q={запрос}&per_page=10`

Заголовок `User-Agent` обязателен. Ответ парсится как JSON. Для неаутентифицированных запросов действует лимит 60 запросов в час с одного IP.

Пример ответа API:
```json
{
  "total_count": 42,
  "items": [
    {
      "login": "torvalds",
      "avatar_url": "https://avatars.githubusercontent.com/u/1024025?v=4"
    }
  ]
}
```

Из него извлекаются поля `login` и `avatar_url`.

## Примеры использования

1. **Поиск:** введите, например, `torvalds` и нажмите «Искать». В списке появятся соответствующие пользователи.
2. **Добавление в избранное:** выделите пользователя в списке и нажмите «Добавить в избранное».
3. **Просмотр избранного:** кнопка «Показать избранное» открывает окно со списком.
4. **Удаление:** кнопка «Удалить из избранного» → выбрать пользователя → подтвердить.
5. **Проверка валидации:** попробуйте нажать «Искать» с пустым полем — появится предупреждение.
